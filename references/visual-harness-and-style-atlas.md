# Visual Harness And Style Atlas

Use this file when the user asks to support many more kinds of images, stronger visuality, famous artist-inspired styles, or a more capable generation harness.

This is not an artist-copying file. It converts art and design history into controllable visual parameters for article diagrams.

## Safety Boundary

Do not prompt "in the style of" a living artist or a contemporary copyrighted illustrator.

For famous historical artists and movements, extract transferable principles:

- composition logic
- line behavior
- color discipline
- shape language
- material feel
- spatial rhythm
- typography relation
- figure abstraction

Do not copy signature works, named characters, exact compositions, or distinctive personal marks.

Good:

```text
strict orthogonal grid, primary color blocks, black divider lines, asymmetric balance
```

Bad:

```text
make it look like a specific Mondrian painting
```

## Supported Output Families

Use this as the widened harness menu before choosing a specific diagram form. The output family is the delivery shape; the diagram form is the logic inside it.

### Article Body Figures

- thesis anchor: one strong metaphor or system map for the central claim
- mechanism diagram: how something works, fails, or changes
- tension diagram: tradeoff, bottleneck, conflict, or constraint
- evidence figure: comparison, scorecard, data-object hybrid, or annotated table
- human consequence figure: role, pressure, waiting, work/life effect, or emotional state
- summary map: end-of-article synthesis, field map, canvas, or route

### Diagram And Infographic Families

- flow / process / swimlane
- layered pyramid / stack / maturity ladder
- matrix / quadrant / scorecard
- map / territory / route / subway / city
- timeline / calendar / before-after sequence
- loop / flywheel / feedback system
- funnel / filter / bottleneck / loss diagram
- cutaway / anatomy / exploded object
- ecosystem / network / dependency map
- small multiples / storyboard / repeated panels
- part-to-whole / nested containers / treemap-like editorial object
- spectrum / scale / slider / balance
- tree / taxonomy / nested hierarchy
- overlap / Venn / Euler, only when overlap is meaningful
- Sankey / alluvial, only when true quantities or weighted flows exist
- service blueprint / frontstage-backstage journey
- causal model / systems model, only when causal evidence exists
- value chain / strategic landscape
- framework canvas / operating model board
- spatial stack / isometric layers / room / terrain / orbit
- object board / field atlas / zukan card

### Editorial And Social Families

- poster-card: one claim, one focal object, short title zone
- cover image: broader mood plus accurate concept, not dense labels
- newsletter header: simple spatial metaphor, restrained label area
- carousel slide: one idea per frame, consistent carrier and labels
- mini comic explainer: 2-4 panels, same camera, one variable changes
- field guide spread: taxonomy, specimens, annotations, side notes
- premium quote figure: one sentence, one visual proof object, no fake attribution

### Technical And Product Families

- architecture explanation without implementation falsehoods
- product teardown / interface mechanism / decision pipeline
- security or failure path / risk boundary / trust perimeter
- API or data flow as object route, swimlane, or cutaway
- operational workflow / service blueprint / workbench state map
- research synthesis / insight board / evidence trail

Reject an output family if it would force invented data, invented causality, a fake screenshot, a fake interface, or a denser figure than the article can support.

### 3. Style Lineage

Choose one or blend two compatible lineages.

Do not blend more than two unless the brief is exploratory.

Lineage is not decoration. It must answer one of these:

- makes the reader path more legible
- makes hierarchy or grouping more visible
- makes the article's mood more accurate
- makes an abstract idea physically memorable
- makes a package feel unified without a mascot

### 4. Abstraction Level

| Level | Use |
| --- | --- |
| A0 literal | product/object must be recognizable |
| A1 simplified | ordinary article diagram default |
| A2 symbolic | concepts as tokens, cards, paths, containers |
| A3 abstract | geometric/emotional relations dominate |
| A4 non-objective | rare; only for mood, cover, or conceptual essay |

Default: A1-A2.

## Historical Style Lineages

Use these as selectable style lineages. They are compatible with diagram forms because each gives a different way to handle space, line, color, or hierarchy.

### Bauhaus / Constructive Workshop

Best for: mechanisms, education, workflows, system thinking, tool diagrams.

Tokens:

- simple geometry
- functional composition
- primary shapes
- visible construction logic
- clean typography
- parts and assembly

Use with: flow, cutaway, canvas, service blueprint, object board, technical plate.

Avoid: decorative nostalgia, faux vintage posters.

Prompt phrase:

```text
Bauhaus-informed constructive workshop diagram, simple geometry, functional layout, visible assembly logic, clean label discipline
```

### De Stijl / Neo-Plastic Grid

Best for: matrices, tradeoffs, hierarchy, constraints, structured arguments.

Tokens:

- strict horizontal/vertical geometry
- black dividers
- primary color accents
- white space as structure
- asymmetrical balance

Use with: matrix, hierarchy, canvas, scorecard, part-to-whole.

Avoid: copying a specific painting; do not overuse red/yellow/blue as decoration.

Prompt phrase:

```text
De Stijl-informed editorial grid, strict orthogonal structure, restrained primary accents, white space as active structure
```

### Constructivist Poster Logic

Best for: urgency, systems under pressure, manifesto-like claims, directional flows.

Tokens:

- diagonal force
- limited palette
- strong contrast
- bold bands
- mechanical rhythm
- directional typography zones

Use with: risk boundary, funnel, causal model, value chain, tension diagram.

Avoid: propaganda cosplay, fake political poster, overbearing red.

Prompt phrase:

```text
constructivist-informed poster logic, diagonal pressure, bold limited palette, mechanical rhythm, clear information hierarchy
```

### Suprematist Geometry

Best for: abstract relationships, thesis anchors, conceptual covers, non-character identity.

Tokens:

- floating geometric forms
- limited colors
- strong negative space
- pure relation over literal scene
- tension through position and scale

Use with: object metaphor, orbit, tension, hierarchy, conceptual map.

Avoid: too abstract for mechanism or data-heavy sections.

Prompt phrase:

```text
suprematist-informed geometric relation, floating primitives, strong negative space, limited color, readable conceptual hierarchy
```

### Cubist / Analytic Faceting

Best for: multiple perspectives, fragmented systems, ambiguity, tradeoff.

Tokens:

- faceted planes
- overlapping viewpoints
- muted palette
- partial object views
- fractured but controlled structure

Use with: cutaway, hidden mechanism, tension, comparison, personal/cultural ambiguity.

Avoid: illegible fragmentation; never use it for exact quantitative charts.

Prompt phrase:

```text
cubist-informed faceted planes, multiple viewpoints, controlled fragmentation, labels on stable facets
```

### Matisse Cutout Logic

Best for: warmth, simple thesis images, emotional essays, social posts, child-readable explainers.

Tokens:

- cut-paper shapes
- lively color contrast
- economy of means
- organic simplified forms
- generous white or colored paper field

Use with: object metaphor, storyboard, room, map, poster-card, soft diagram infographic.

Avoid: decorative leaf shapes that do not carry meaning.

Prompt phrase:

```text
cut-paper composition, lively color contrast, organic simplified shapes, economy of means, warm editorial clarity
```

### Ukiyo-e / Hokusai-Informed Composition

Best for: journeys, pressure waves, natural forces, market/weather metaphors, dramatic foreground/background relation.

Tokens:

- bold contour
- flat color planes
- cropped perspective
- rhythmic wave or route forms
- foreground force vs distant anchor

Use with: terrain, map, journey, pressure field, risk wave, timeline.

Avoid: Japanese pastiche, faux calligraphy, copying the Great Wave.

Prompt phrase:

```text
ukiyo-e-informed flat color composition, bold contour, cropped perspective, rhythmic force shape, distant anchor
```

### Kandinsky / Musical Geometry

Best for: emotion systems, attention, rhythm, abstract agent loops, conceptual tension.

Tokens:

- circles, arcs, angled lines
- color as emotional weight
- musical rhythm
- overlapping abstract signals
- dynamic balance

Use with: orbit, feedback loop, emotional curve, network, tension map.

Avoid: chaotic abstraction with no reader path.

Prompt phrase:

```text
musical geometric abstraction, circles arcs and angled lines, emotional color weight, disciplined reader path
```

### Paul Klee / Symbolic Micro-World

Best for: gentle complexity, learning, personal essays, culture, small systems with charm.

Tokens:

- small signs
- playful symbolic grids
- warm irregular geometry
- tiny worlds
- poetic but legible marks

Use with: field atlas, object board, storyboard, taxonomy, room/stage.

Avoid: childish doodle, cryptic symbols with no legend.

Prompt phrase:

```text
symbolic micro-world, warm irregular geometry, small signs, poetic but legible marks, disciplined labels
```

### Josef Albers / Color Interaction

Best for: comparison, perception, tradeoffs, attention, nested states.

Tokens:

- nested squares or fields
- adjacent color interaction
- flat planes that advance/recede
- strict simple composition

Use with: matrix, part-to-whole, nested hierarchy, evidence comparison, spectrum.

Avoid: pure color study with no article meaning.

Prompt phrase:

```text
color-interaction field, nested planes, disciplined adjacent colors, simple geometry, direct labels
```

### Calder / Mobile Balance

Best for: dependency, dynamic equilibrium, tradeoff, ecosystem, fragile balance.

Tokens:

- suspended shapes
- thin rods or threads
- balanced weights
- movement implied by spacing
- playful but calibrated structure

Use with: ecosystem, causal model, tradeoff scale, orbit, dependency map.

Avoid: circus whimsy; do not copy specific mobiles.

Prompt phrase:

```text
mobile-like balance diagram, suspended shapes, thin connectors, calibrated equilibrium, playful but precise structure
```

### Swiss / International Typographic Clarity

Best for: serious analysis, business, product, technical, evidence-heavy articles.

Tokens:

- grid
- asymmetric layout
- sans-serif typography
- objective structure
- flush-left label discipline
- large whitespace

Use with: scorecard, matrix, timeline, report figure, evidence table.

Avoid: sterile corporate slide; add one focal surprise.

Prompt phrase:

```text
Swiss-style information design, strict grid, asymmetric layout, sans-serif label discipline, objective clarity, generous whitespace
```

### Art Nouveau Organic Line

Best for: growth, flow, ecology, life systems, emotional continuity.

Tokens:

- long sinuous organic line
- asymmetry
- nature-inspired curves
- flow and transition

Use with: journey, growth loop, garden metaphor, emotional curve.

Avoid: ornamental overload, decorative vines, low readability.

Prompt phrase:

```text
organic-line editorial diagram, long sinuous carrier, asymmetrical flow, restrained nature-inspired curves, readable labels
```

### Art Deco / Streamlined Geometry

Best for: premium social cards, hierarchy, city/market, aspiration, bold summaries.

Tokens:

- stepped geometry
- symmetry or near-symmetry
- metallic or deep accent feeling
- streamlined rays
- poster-like confidence

Use with: pyramid, ladder, poster-card, value chain, city map.

Avoid: luxury decoration that hides the argument.

Prompt phrase:

```text
streamlined geometric editorial poster, stepped forms, confident symmetry, restrained premium accents, clear information hierarchy
```

### Risograph / Editorial Print

Best for: newsletters, culture, social cards, approachable essays.

Tokens:

- limited ink colors
- slight texture
- overprint feeling
- friendly imperfection
- flat shapes

Use with: object metaphor, storyboard, field guide, comparison.

Avoid: muddy low-contrast labels.

Prompt phrase:

```text
limited-ink editorial print, subtle overprint texture, flat shapes, warm imperfection, crisp label zones
```

## Style Lineage Pairings

Good pairs:

- Swiss + PrismGrid: clear analytical diagrams
- Bauhaus + Technical Plate: mechanisms and assembly
- De Stijl + Matrix: decision spaces
- Matisse Cutout + Soft Diagram: warm public explainers
- Ukiyo-e + Terrain: pressure, journey, market/weather metaphor
- Klee + Field Atlas: gentle taxonomy and learning
- Albers + Matrix/Part-to-whole: perception and comparison
- Calder + Ecosystem: balance and dependency
- Constructivist + Risk Boundary: urgency and pressure
- Art Nouveau + Growth Loop: organic continuity
- Art Deco + Poster-Card: confident summaries and premium covers
- Risograph + Newsletter Header: approachable essays and cultural commentary
- Cubist + Hidden Mechanism: multiple viewpoints without pretending exact data

Bad pairs:

- Constructivist + Art Nouveau + Klee all together
- Swiss clarity with dense decorative pattern
- Cubist fragmentation for precise data
- Art Deco luxury for safety warnings
- Kandinsky abstraction for procedural instructions
- Hokusai-informed waves for unrelated "make it pretty" decoration
- Albers color interaction when color does not encode comparison or state
- Matisse cutout logic when organic shapes become meaningless confetti

## Harness Procedure

For every non-trivial image:

1. Name the article job.
2. Select one output family.
3. Select one diagram form inside that family.
4. Select one spatial mode.
5. Select one style lineage because it improves comprehension, not because it is famous.
6. Select material, line behavior, color system, and composition engine.
7. Lock typography mode and human presence.
8. Write the anti-copy boundary.
9. Reject the plan if the lineage makes the diagram less accurate.

## QA

Accept if:

- the style lineage improves comprehension or memory
- the selected style does not override the diagram form
- the image is not copying a specific artist or artwork
- the visual harness is internally coherent
- the label and token systems still work

Reject if:

- it says only "in the style of X"
- it imitates a living artist or a copyrighted contemporary illustrator
- it uses a famous artwork composition as the layout
- style makes labels harder to read
- style creates fake data, causality, hierarchy, or emotion
- the image is attractive but no longer an article figure
